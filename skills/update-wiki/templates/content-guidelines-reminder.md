# Content Guidelines — Quick Reminder

When editing existing wiki page sections, maintain consistency with these rules:

- Introduction format: 1-2 paragraphs, up to 300 characters
- Use H2/H3 headings for logical sections
- Mermaid diagram rules: `graph TD` only, max 3-4 word nodes, sequence diagram conventions
- Tables for summarizing APIs, config, data models
- Code snippets: optional, from source files only
- Source citations: `Sources: [filename.ext:start-end](<relative_file_path>/filename.ext#Lstart-Lend)`
- Technical accuracy: derive all information solely from source files
